## Getting Started

### Installation

1. Get a free API Key at [OpenWeather](https://openweathermap.org/api)
2. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/Weather-app-React-native.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start Expo app
   ```sh
   npm start
   ```
